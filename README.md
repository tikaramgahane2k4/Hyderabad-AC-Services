# Hyderabad AC Services Website

## 📌 Project Overview

This project is a complete redesign of the Hyderabad AC Services website using modern web technologies.

The goal is to build a clean, responsive, and brand-compliant website based on the client’s provided content and brand guidelines.

---

* CSS / Tailwind (if used)
* Git & GitHub for version control

## 👥 Team Workflow

### Branch Strategy

* `main` → Production-ready code
* `dev` → Development branch
* `feature/*` → Individual features

### Workflow

1. Create a feature branch:

   ```
   git checkout -b feature/feature-name
   ```

2. Work and commit:

   ```
   git add .
   git commit -m "Added feature"
   ```

3. Push branch:

   ```
   git push origin feature/feature-name
   ```

4. Create Pull Request → `dev`

---

## ⚠️ Rules

* Do NOT push directly to `main`
* Always use feature branches
* Follow brand guidelines strictly
* Keep commits clean and meaningful

## 📦 Setup Instructions

Clone the repository:

```
git clone https://github.com/noorsetia/Hyderabad-AC-Services.git
cd Hyderabad-AC-Services
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
